# go-crypt-sync
v0.0.1

## MVP

Everything needs to be done in memory. No writing files unless with the -o flag specified.
This is so that we don't accidently leave an unencrypted file on the disk.

### View
- Read file from gcs bucket (encrypted)
- Decrypt file 
- View file

### Add
- Read file from gcs bucket (encrypted)
- Decrypt file 
- Add to file
- Encrypt file
- Write file
- Upload to gcs bucket
